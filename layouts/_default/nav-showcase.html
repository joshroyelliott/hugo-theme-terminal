Navigation Bar Components

The navigation bar system in this theme is designed to mimic a TUI (Text User Interface) application's toolbar or status line. It provides flexible configuration for creating various navigation bar layouts.

Components Overview

Basic Components

  • Nav Container: The dark bar that spans the page width
  • Nav Lists: Groups of navigation elements within the container
  • List Title: An optional title tab for each list (dark background)
  • List Items: The clickable navigation items (light background)

Configuration Options

  • direction: Controls chevron orientation ("right" or "left")
  • float: Controls list positioning within container ("left" or "right")
  • flatten: Determines if edges against boundaries should be flat (true/false)
